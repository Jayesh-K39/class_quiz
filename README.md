# Class Quiz Pro
An open-source bidirectional quiz app.

This is a simple application that aims to solve one specific painpoint:
**Assessing student's in-class performance** (especially when the strength of the class is high)

It allows the teachers to start a crete their own quizzes, and stores those quizzes for later use as well.

## Additional Features:
- Student LeaderBoard 
- Pause/cancel/restart quiz
- Secured (HTTPS instead of HTTP)
